This directory contains the files to build XLI XPRESS library.

This library allows lp_solve to read and write model files in the XPRESS lp format.

See http://artelys.com/fr/pdf/optimizer.pdf
for a description about the XPRESS lp format.

To build the program under Windows with the Visual C/C++ compiler, use cvc6.bat (also works for VS.NET)
To build the program under Linux/Unix, use sh ccc
